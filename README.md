# Smarket Application

Smarket is a multi-vendor marketplace web application built using Flask and SQLAlchemy.

## Features

- Buyer, Seller, and Admin roles
- Product listing and approval system
- Shopping cart and wishlist
- Real-time chat using Socket.IO
- Notifications system
- Order management and tracking
- Payment workflow
- Product reviews and feedback
- Admin analytics dashboard
- Seller revenue reports

## Technologies Used

- Flask
- SQLAlchemy
- Flask-Login
- Flask-SocketIO
- HTML/CSS/JavaScript
- SQLite / MySQL

## Installation

```bash
pip install -r requirements.txt
python3 setup.py
python3 app.py
```

## Run Application

Open:

http://127.0.0.1:5000
